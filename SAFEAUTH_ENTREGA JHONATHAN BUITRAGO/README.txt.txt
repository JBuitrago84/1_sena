Proyecto: SafeAuth

Este proyecto contiene un sistema básico de gestión de usuarios desarrollado en Java con conexión a MySQL mediante JDBC.

Contenido:
- Carpeta proyecto: código fuente desarrollado en NetBeans.
- Carpeta base_datos: archivo SQL para importar la base de datos.

Instrucciones:
1. Importar la base de datos en phpMyAdmin.
2. Abrir el proyecto en NetBeans.
3. Ejecutar la clase Main.java.

El sistema permite realizar operaciones CRUD:
- Insertar usuarios
- Consultar usuarios
- Actualizar usuarios
- Eliminar usuarios
